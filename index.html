Segmap is a landscape image segmentation service based on Deep Convolutional Neural Networks. The project is aimed to Earth surface monitoring to address changes quickly and efficiently

Use the form below to upload and segment your image. A file should contain 3 channels (RGB) with a pixel value range between 0 and 255.
